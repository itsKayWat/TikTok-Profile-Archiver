Privacy Policy for TikTok Backup Tool

Introduction

The TikTok Backup Tool is a utility designed to help users create local backups of TikTok profiles, including videos, profile information, and associated metadata. This privacy policy explains how the tool handles data and protects user privacy.

Data Collection & Storage

This tool operates with complete privacy:

  • All downloaded content is stored locally on your device
  • No data is transmitted to external servers
  • No personal information is collected
  • No analytics or tracking is performed
  • No cookies are used

TikTok Interaction

The tool interacts with TikTok in the following ways:

  • Uses Chrome browser automation to access public TikTok content
  • Downloads publicly available videos and information
  • Respects TikTok's robots.txt and rate limits
  • Only accesses content that is publicly viewable

Local Storage

Content is stored locally with the following considerations:

  • All downloads are saved to your local device
  • Organized in a clear directory structure
  • You have complete control over the stored files
  • You can delete the backed-up content at any time

Browser Integration

The tool requires Chrome browser access to:

  • Navigate TikTok pages
  • Extract video URLs and metadata
  • Download content through yt-dlp
  • No browser history or personal data is accessed

User Control

Users maintain full control over:

  • Which profiles to backup
  • What content to download
  • Where files are stored
  • When to delete backed-up content

Legal Compliance

Important legal considerations:

  • Only use this tool for personal archiving
  • Respect copyright and intellectual property rights
  • Follow TikTok's terms of service
  • Obtain permission before sharing backed-up content

Contact Information

For questions about this privacy policy or the tool's data handling, please reach out through GitHub or email: info@kaywat.me
